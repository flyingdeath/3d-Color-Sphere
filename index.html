3d-color-sphere

3d Color Sphere with, Color Schemas builtin

View on GitHub Download .zip Download .tar.gz

Website http://flyingdeath.github.io/3d-Color-Sphere

This project is 3d Color Sphere with, Color Schemas builtin, Implementing HSV Think of being inside 3d Color Sphere where HSV are mapped to a Polar coordinate system pole theta phi.

Geometric Polygon Type Schemas (Color Theory) : Use mouse wheel to rotate geometry in the x y plan

Complementary (size slider) Analogous (size slider) Triadic (size slider) Split-Complementary (size slider) Rectangle (width height slider) Square (width height slider) Polygonal (click to create polygon size slider )

Value Gradient (value slider) Random swatch generator (value slider) / Random Value 5 swatch Color Patterns

Using HTML, CSS and Javascript, YUI, Three JS, WebGL.

WebGL Needed to work, so if it doesn't work on your system try it on a WebGL Video Card. Nvidia and ATI Both make Video Card that support WebGL.
